Project Overview

This project is an interactive HR Analytics dashboard built using Power BI to analyse workforce data, attrition trends, salary distribution, and employee experience.

The objective of this project is to transform raw HR data into meaningful insights and create a professional dashboard for decision-making.

1)Dataset Source:

The dataset used in this project is a sample dataset taken from a YouTube tutorial for learning and practice purposes.

2)Key Insights:

Workforce distribution by age and gender

Attrition analysis by job role and experience

Salary distribution by qualification

Employee growth trend over time

Leave balance analysis

Experience distribution across employees

3)Tools & Technologies:

Power BI

Data Modeling

DAX

Data Visualization

4)Features:

Interactive dashboard

KPI cards for headcount, salary, attrition

Trend analysis visuals

Role-based and experience-based insights


 5)Learning Outcome
